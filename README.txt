gezgin1.rar isimli jar dosyası üzerine tıklayınız. İlk çıkan ekranda sizden kaç şehir gezileceği bilgisi istenmektedir. 
Gezmek istediğiniz şehir sayısını giriniz ardından devam butonuna tıklayınız (Kocaeli hariç zaten başlangıç şehiri olarak Kocaeli belirlenmiştir.). 
İkinci gelen ekranda gezgin kargonun gitmek istediği şehirlerin isimlerini giriniz. Bu isimleri girerken lütfen Türkçe karakterler kullanmamaya özen gösteriniz. 
Şehirlerin isimleri sehir.txt dosyasından okunduğu için girilen şehir isimleri de aynı formatta olmalıdır. Örneğin : Ağrı değil Agri , Balıkesir değil Balikesir gibi. 
Gezmek istediğiniz şehirlerin isimlerini girdikten sonra devam butonuna basınız. Karşınıza beş tane GreaphStream ekranı gelecektir (eğer gezmek istediğiniz şehir sayısı ikiden fazla ise) bu ekranlara ve gösterilen ok yönlerine bakarak en kısa 5 yolu görebilirsiniz. 
Eğer diğer alternatif yolları da görmek istiyorsanız tum_yollar.txt dosyasını açınız.



Eğer .jar dosyasını çalıştaramıyorsanız veya JNİ error mesajını alıyorsanız, java ve javac version aralarında uyumusuzluk olabilir.
Bu proje kapsamında kullandığımız versiyonlar:

java version "1.8.0_251"
Java(TM) SE Runtime Environment (build 1.8.0_251-b08)
Java HotSpot(TM) 64-Bit Server VM (build 25.251-b08, mixed mode)

javac 14.0.1
